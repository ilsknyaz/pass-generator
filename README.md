# Генератор паролей
Князев МПБ-301-О-01
Генератор безопасных паролей длиной до 32 символов с использованием настроек регистра и наличия специальных символов. Для тех, кто желает сохранить свои данные под надежной защитой.

## Стек
- HTML, CSS, JavaScript
- FontAwesome

## Быстрый старт
```bash
cd pass-generator
firefox [chrome] index.html
```

или

В папке проекта ```pass-generator``` открыть файл ```index.html``` в любом браузере.
